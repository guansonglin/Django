# Django项目

#### 介绍
一个简单的用户管理系统，搭配数据库

#### 软件架构
软件架构说明采用python + MySql + Django所做成的前后端分离的系统。可以用于毕设，但是需要自己加以修改。


#### 安装教程

1.  python > 3.7.2
2.  mysql > 5.7.37
3.  django > 3.2.16
